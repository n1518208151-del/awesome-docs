## AIBOX (AX8850 AIBOX)

### 产品亮点与规格

| 字段 | 内容 |
| --- | --- |
| 芯片 | AX8850 |
| 算力 | 24 TOPS@INT8 |

| 存储 | 8GB/32GB eMMC |
| 显示 | 标准（A TYPE）HDMI *2 |
| 网络 | RJ45(10/100/1000M) *2 |
| USB 与扩展 | USB3.0 TYPE-A *1, USB2.0 TYPE-A *1, RS232, RS485|

### 适用场景

- 智慧园区：解决禁区闯入，岗位无人值守，出入口人员聚集，车辆进出管理等问题。
- 工业安全：解决未戴安全帽，明火烟雾，禁止吸烟区域违规等问题。
- 商业门店与仓储：解决客流统计，排队拥堵，异常徘徊等问题。
- 交通与停车：解决车辆进出识别，禁停区域占用，逆向行驶，车辆流量统计等问题。
